## ID do Bug: BUG-001 ##

**Titulo:** Usuário consegue realizar cadastro com o mesmo email já cadastrado.

**Severidade:** Alta

**Prioridade:** Alta

**Ambiente:** Windows, OperaGX 126.0.5750.56 (64-bits)

**Pré-condição:** Usuário não deve estar cadastrado

**Passos para reproduzir:** 
  1. Estar na Home Page.
  2. Clicar no botão [Cadastro].
  3. Prencher os campos de nome, e-mail, senha e confirmar senha.
  4. Clicar no botão [Enviar].

**Resultado Esperado:** Usuário não deve conseguir realizar o cadastro utilizando um e-mail já cadastrado.

**Resultado Atual:** Usúario consegue realizar o cadastro utilizando um e-mail já cadastrado.
